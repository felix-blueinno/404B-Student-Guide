404B Student Setup Guide

Requirements

1. GitHub Account

You need a GitHub account to use GitHub Classroom. If you don't have one, you can create one for free here.

2. Git Installation

You need to have Git installed on your computer. If you don't have Git installed:

2.1 Git Installation for Mac

  1. Go to HomeBrew and copy the command under "Install Homebrew".

  2. Press command + space to open Spotlight Search and type "Terminal" and press enter.

  3. Paste the command you copied from HomeBrew into the Terminal and press enter.

    Note

    You may be prompted to enter your password. If so, enter your password and press enter.

    You will not see your password as you type it. This is normal.

  4. Press Enter again to confirm the installation.

  5. Once the installation is complete, enter:

    brew install git

2.2 Git Installation for Windows

  1. Go to Git Homepage and click on the "Windows" button under "Downloads".

  2. Download the "64-bit Git for Windows Setup".

  3. Once the download is complete, open the file and follow the instructions to install Git.

3. Python Installation

You need to have Python installed on your computer. If you don't have Python installed:

3.1 Python Installation for Mac

  1. Visit https://www.python.org/.

  2. Click Download.

  3. Introduction: Continue

  4. Read Me: Continue

  5. Licence: Continue >> Agree

  6. Installation Type: Install >> Password

  7. Close window when completed:

  8. In Applications >> Python 3.XX, find Install Certificates.command. Double click to execute it and wait for Process completed.

  9. In Applications >> Python 3.XX, find Update Shell Profile.command. Double click to execute it and wait for Process completed.

  10. In Spotlight, find terminal.

  11. Make sure keyword python is working in terminal:

    python3 --version

    If error, find teacher.

3.2 Python Installation for Windows

  1. Visit https://www.python.org/.

  2. Click Download.

  3. Make sure Add Python 3.X to PATH is ticked, before clicking Install Now.

  4. While installing, click "ALLOW" if permission dialog pops up.

  5. Close the installer when completed.

  6. Press Win + R, ⇒ “cmd” ⇒ OK

  7. Make sure keyword python is working in command line:

    python --version

  8. If error, start over from step 3.

4. Visual Studio Code Installation

You need to have Visual Studio Code installed on your computer. If you don't have Visual Studio Code installed:

Go to Visual Studio Code Homepage and click on the "Download for Mac" or "Download for Windows" button.

4.1 Visual Studio Code Installation for Mac

  1. https://code.visualstudio.com/
  2. Click “Download”
  3. Extract the downloaded .zip file
  4. (Optional) Move Visual Studio Code from Downloads folder to Application folder

4.2 Visual Studio Code Installation for Windows

  1. Visit https://code.visualstudio.com/
  2. Click “Download”
  3. Accept agreement
  4. Install location
  5. Create shortcut in Start Menu
  6. Create shortcut on Desktop
  7. Click “Install”.
  8. Dismiss installer when completed

5. VS Code Extensions Installation

  1. Open VS Code.
  2. Install “Python” extension.
  3. Install "LiveShare" extension.

6. GitHub Classroom Invitation

  1. Open the invitation link provided by your teacher.
  2. Click on the "Accept the assignment" button.
  3. Click on the "Open in Visual Studio Code" button.

7. Submitting Your Work

  1. Make sure the assignment repository is opened in VS Code.

  2. Make sure you have completed all the tasks.

  3. (First time only)
    Use Command + J to open the Terminal tab and config your git username and email:

    git config user.name "Your Name"
git config user.email "Your GitHub Email"

  4. Click on the "Source Control" icon on the left.

  5. Enter a commit message and click on the "Commit" button.

  6. Click on the "Sync Changes" button.
